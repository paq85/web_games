How to Play

Core Gameplay

Slide all tiles on the 4x4 grid in one direction at a time. Tiles with the same value merge into one, doubling their value. After every move a new tile (value 2 or 4) spawns in an empty cell.

Goal: Create a tile with the value 2048 (Classic mode) or reach the highest score possible (Endless mode).

Controls

Arrow keys or W A S DSlide tiles
Esc / PPause
ZUndo last move
MMute audio
SwipeSlide tiles (touch)
DragSlide tiles (mouse)

Special Tiles

★WildcardMerges with any tile, taking its value
💥BombCannot merge; destroys both tiles on collision
🛡ShieldCannot merge for one move, then becomes normal
×2MultiplierDoubles adjacent merge values, then consumed
⚡Fusion CoreTriples merge result when merging with equal-value tile

Grid Zones

Gravity WellPulls tiles downward after each move
Frozen ZoneTiles locked in place, immune to slides
Boost ZoneDoubles merge values within the zone
Swap ZoneAllows one free tile swap per move

Power-ups

↩UndoRevert the last move
✂SplitSplit a tile into two halves
☢NukeDestroy all tiles of a chosen value
❄FreezeNo new tile spawns on next move
⇄SwapSwap any two tiles on the grid
⚓StabilizePrevent the next grid mutation

Tap a power-up slot, then tap a tile to target it. Earn charges as you score points.

Grid Mutations

Between moves the grid may randomly shift a row, shift a column, or rotate a quadrant. Mutations become more frequent as your level increases. Use the Stabilize power-up to prevent them.

Combo System

Consecutive moves that cause merges build a streak multiplier: 2 merges = 1.5x, 3 = 2x, 4 = 3x, 5+ = 5x. Any move without a merge resets the streak.

Game Modes

ClassicReach a 2048 tile to win. All mechanics active.
EndlessNo win target. Play until no moves remain.
ChallengeComplete specific objectives for star ratings.
Daily PuzzleFixed-seed puzzle with 3 attempts per day.
