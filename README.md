# PAQ85 Web Games

A collection of retro-style web games built with HTML5 Canvas and vanilla JavaScript.

## Play

- **Live demo:** [https://paq85.github.io/web_games/](https://paq85.github.io/web_games/)
- **Local:** Open [`index.html`](index.html) in any modern browser. No build tools or server required.
