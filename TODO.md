# Game Ideas - Backlog

## Existing Games
- **Pong** - Pixel-retro arcade Pong (2-player + AI)
- **Flappy Bird** - Tap-to-fly obstacle game
- **Tic Tac Toe** - Classic 2-player strategy
- **UGH!** - Prehistoric helicopter transport (10 levels, hazards, fuel management)

## Candidate Games

| Game | Category | Why It Fits |
|------|----------|-------------|
| **Breakout / Arkanoid** | Brick breaker | Classic paddle-and-ball mechanics. Power-ups, level design, satisfying progression. No overlap with existing games. |
| **Space Invaders** | Classic shooter | Top-down wave-based shooter. Scoring, increasing difficulty, alien formations. Distinct from all existing games. |
| **Snake** | Arcade strategy | Growing snake, food collection, avoid walls/self. Simple rules, deep skill ceiling. |
| **2048** | Modern puzzle | Sliding-tile grid puzzle. No canvas needed, pure logic. Very different from the arcade-heavy lineup. |
| **Tetris-like** | Block puzzle | Falling blocks, line clears, increasing speed. Deep and timeless gameplay. |
| **Lode Runner** | Puzzle-platformer | Levels, ladders, digging, gold collection, enemies. More complex design with level editor potential. |
| **Frogger** | Crossing game | Traffic and river crossing. Simple concept, level progression, retro feel. |
| **Doodle Jump** | Vertical climber | Tap to jump upward on platforms. Simple controls, increasing difficulty, mobile-friendly. |
| **Centipede** | Arcade shooter | Player shoots segments of a moving centipede that splits. Mushrooms as obstacles. Unique movement patterns. |
| **Chip's Challenge** | Puzzle-adventure | Grid-based puzzle with items, switches, teleporters. Level progression with diverse mechanics. |
